# Directory Lister Script

Simple Bash script that prompts the user for a directory path, checks if the directory exists, and saves the listing of its contents to a file called `directory_content.txt`.

## How to use

1. Clone/download this repository  
2. Make the script executable:  
  ```bash
  chmod +x list_dir.sh
  ```
3. Run the script:  
  ```bash
  ./list_dir.sh
  ```
4. When prompted, enter the path of the directory you want to list  
5. If the directory exists, the contents will be saved in `directory_content.txt`
